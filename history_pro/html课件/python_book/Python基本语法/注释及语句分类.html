注释

注释就是：注解，解释。

主要用于在代码中给代码标识出相关的文字提示(提高代码的可读性) 或 调试程序。Python中注释分为两类：

1.单行注释： 单行注释以 # 号开头，在当前行内，# 号后面的内容就是注释内容 2.多行注释： 多行注释使用两个 ''' 或两个 """ 将需要注释的内容包裹起来

Python语句分类

Python语句分为两类：单行语句 和 代码块/代码组

* 单行语句指的是一行Python代码。
* 代码组指的是特定的语言结构。
